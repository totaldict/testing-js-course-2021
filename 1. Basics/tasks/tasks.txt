Задание 1:
  1. Инициализировать новое node.js-приложение

  2. Создать класс с методом removeFalsyValues
    - Пример: [45, '', 0, 'hello', 4.7, false] => [45, 'hello, 4.7]
    - Falsy-значения: '', 0, false, null, undefined, NaN

  3. Тестовые сценарии:
    - Falsy-значения удаляются из непустого массива
    - Аргумент является массивом

Задание 2:
  С использованием TDD написать метод получения токена из заголовка Authorization

  1. Для пустой строки должен возвращаться null
  2. Для "нормальной" строки должен возвращатсья токен
  3. Длина токена не должна быть меньше 10 символов
  4. Токен должен содержать только буквы английского алфавита и цифры 

  Пример входных данных: 'Bearer vCVVCHjTylu2lMtEf5wjljd5VvBkuizKYGITwAoV75XXj5uiv30MLYtINF'
  Пример выходных данных: 'vCVVCHjTylu2lMtEf5wjljd5VvBkuizKYGITwAoV75XXj5uiv30MLYtINF'
